# 批量独立 Chrome 登录窗口工具

这是一个 Windows 桌面小工具，用于按 Excel 名单批量生成多个互相隔离的 Chrome 登录窗口。适合需要为多名学生、家长或账号分别登录同一个网站的场景。

## 功能概览

- 从 `学生电话表.xlsx` 读取 `姓名` 和 `手机` 两列。
- 为每个人生成一个独立 Chrome 数据目录。
- 每个账号的 Cookie、登录状态、缓存相互隔离，不会串号。
- 自动生成姓名和手机号标牌页，方便核对当前窗口属于谁。
- 自动按 5 个账号一组生成分组启动器，减少一次打开过多窗口导致卡顿。

## 文件说明

| 文件 | 作用 |
| --- | --- |
| `学生电话表.xlsx` | 名单模板，只需要填写 `姓名` 和 `手机` 两列 |
| `双击生成登录窗口.bat` | 双击入口，负责启动 PowerShell 生成器 |
| `generate_login_windows.ps1` | 核心生成脚本 |
| `使用说明_先看.txt` | 给普通使用者看的简明说明 |

运行后会自动生成 `生成结果_登录窗口` 文件夹，里面包含：

- `acc01.bat`、`acc02.bat` 等单账号启动器
- `group1_no01-05.bat` 等分组启动器
- `data/accNN` 独立 Chrome 登录数据目录
- `labels/accNN.html` 姓名手机号标牌页
- `对照表.txt`

## 使用方法

1. 打开 `学生电话表.xlsx`。
2. 从第 2 行开始填写姓名和手机号。
3. 保存并关闭 Excel。
4. 双击 `双击生成登录窗口.bat`。
5. 打开生成出来的 `生成结果_登录窗口`。
6. 双击 `group1_no01-05.bat`、`group2_no06-10.bat` 等分组启动器。

## 修改目标网址

目标网址在 `generate_login_windows.ps1` 顶部：

```powershell
$url = "https://icc.cffpd.org.cn/activity?competitionType=theme"
```

需要换网站时，修改这个变量后重新双击生成即可。

## 工作原理

Chrome 支持通过 `--user-data-dir` 指定用户数据目录。这个工具为每个账号创建一个独立目录，例如：

```text
data/acc01
data/acc02
data/acc03
```

每个目录保存自己的登录状态，所以多个账号可以在同一台电脑上同时打开，并且互不影响。

## 注意事项

- 建议一次只打开 5 个窗口，电脑更稳定。
- 不要删除 `data` 文件夹，否则对应账号会重新变成未登录状态。
- 想重置某个账号时，关闭对应窗口，删除 `data/accNN`，再重新打开对应启动器。
- 本工具适用于 Windows，需要安装 Google Chrome 和 Microsoft Excel。
